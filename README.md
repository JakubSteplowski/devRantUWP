If you've found a **bug** or want to request a new **feature** for devRant unofficial UWP, just create a new issue.

For **bug reports**, please provide:
- Version of the app
- Device type (Desktop, Mobile, HoloLens, Xbox One, Surface Hub)
- Version of the OS
- Clear steps (if known) to reproduce the issue
- A screenshot (if available)

Thanks in advance! :)
